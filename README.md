# Forgotten_Kingdom
## Projekt indítása
### Legelőszöri indítás
A VueJs projekt mappába:
```
npm build
```
A generált dist mappát át kell húzni a NodeJS gyökér könyvtárába

A NodeJS gyökér könyvtárba a következő parancsal indul el az app:
- Nodemonnal
```
npm start
```
- Nodemon nélkül
```
node index
```
