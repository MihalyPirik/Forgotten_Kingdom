# Forgotten_Kingdom

## Projekt indítása

### Backend

```
npm install
```

.env fájl létrehozása a .env.example segítségével, változók beállítása

```
npm start
```

### Frontend

```
npm install
```

```
npm run build
```

Frontend/WebPage/dist mappába lévő index.html fájlt megnyitni live szerverben
